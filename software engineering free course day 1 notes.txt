Overview software engineering 

bidang profesi yang mendalami cara-cara pengembangan perangkat lunak termasuk pembuatan, pemeliharaan, managemen organisasi pengembangan perangkat lunak 

Common misconceptions 

"Coding is just writing code, not so creative"
"Communication is not important for programmers."
"Testing is not so important than coding"
"The fastest coder is the best programmer"
"Too many bootcamps. Too many developers and software engineers. The job market must be small by now."

tugas-tugas software engineer

1. mendesain dan merawat sistem software
2. testing dan evaluasi program software baru
3. Mengoptimasi speed dan scalability software
4. Menulis dan menguji kode pemrograman
5. Memberikan konsultasi ke klien, sesama engineer, spesialis computer security dan stakeholder lainnya

Technical Skill 
1. Memahami tipe data, struktur, data, dan algoritma
2. Menguasai bahasa pemrograman
3. Memahami penggunaan dasar Operating System 
4. Memahami database dan berbagai konsepnya 
5. Memahami networking serta menguasai encryption dan authentication/authorization
6. Mampu melakukan unit, integration, functional, dan end-to-end testing
7. Memahami project/product management serta software development life cycle

Soft Skill
1. Komunikasi yang baik
2. Mampu bekerja sama dengan orang lain
3. Attention to detail dalam coding
4. Memiliki proses logis dalam menyelesaikan masalah.

Front-Eng Development

Pengembangan kode yang menciptakan elemen visual dari perangkat lunak, aplikasi, atau situs web. Bahasa program front end development termasuk HTML, CSS, dan Javascript
1. HTML (Hypertext Markup language) membantu menyusun struktur website 
2. CSS (Cascading stlye sheets) membantu mendesain halaman website
3. JS (Java Script) untuk menciptakan interaksi yang lebihb dinamis 

Software Development Lifecycle 

Proses pembuatan dan pengubahan sistem serta model dan metodologi yang digunakan untuk mengembangkan sistem rekayasa perangkat lunak 

1. Planning & Analysis 
2. Design 
3. Implement (or Code)
4. Testing & Integration 
5. Deployment 
6. Maintenance
